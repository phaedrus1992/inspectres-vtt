# Introduction

InSpectres is a tabletop role-playing game where players run a paranormal investigation agency.

This repository is for tools to make it easier to set up and run a long-running campaign.

It contains tools for researching and designing mysteries and enemies, managing the agency, and tracking any interesting characters that come and go while playing.
It also contains a plugin for Foundry VTT for running an InSpectres campaign in Foundry.

# Reference Documentation

PDFs of the rulebook for InSpectres is in the `reference/` folder, as well as the output of `pdftotext` on the rulebook.
